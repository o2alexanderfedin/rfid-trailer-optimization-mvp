read-your-writes consistency (the operational twin must never read stale observation state). The tag registry
turns a raw `tagId` into a `packageId` so RFID evidence can be attributed; an unmapped tag is logged, never
an exception (it is not a package).

checkpoints, preserving the FND-04 truncate+replay equivalence.
