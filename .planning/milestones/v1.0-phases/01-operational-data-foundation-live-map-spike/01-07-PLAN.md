PITFALLS P10 / Integration Gotchas — MANDATORY:
- Create the `ol/Map` exactly once, held in a `useRef`; never let React re-render the map node.
- Use ONE reused `ol/source/Vector` per logical layer (hubs, routes, trailers). For trailer updates, mutate existing feature geometry in place (`feature.getGeometry().setCoordinates(...)`) — NEVER rebuild the source each tick.
- Set `crossOrigin: 'anonymous'` on the OSM source; tiles over HTTPS; respect OSM tile usage policy.

</openlayers_discipline>
