| T-03-21 | Denial of service (alert flood) | exception feed flooding the demo | mitigate | above-threshold detection + dedupe (Plan 06) + the false-positive-rate KPI surfaced and asserted-low in the integration test |
| T-03-22 | Elevation of privilege | mutation via the new routes | mitigate | all new routes are read-only; the only writer remains the sim/detector via the event store |
