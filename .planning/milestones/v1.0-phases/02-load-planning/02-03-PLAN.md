CANONICAL DECISIONS (from 02-CONTEXT.md — these are LOCKED, encode them exactly):
- depth 0 = rear (easiest access) → depth N = nose. Zone labels derived from depth ranges
  (lowest third = rear, middle third = middle, highest third = nose).
- unloadOrder(hub) = its stopIndex in the remaining route; LOWER = unloaded sooner = belongs nearer rear.
- Invariant: `unloadOrder(A) < unloadOrder(B) ⟹ depth(A) ≤ depth(B)`.
- Blocker predicate (LOAD-04 preview, defined here so validator + tests share it):
  `isBlocker(target, other)` ⟺ `other.depth < target.depth` (closer to rear) AND `unloadOrder(other) > unloadOrder(target)` (unloads LATER). Same unloadOrder ⇒ NOT a blocker (strict `>`).
